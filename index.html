Fun Facts

Test-web Tim Berners-Lee and his first web page

The world's first web page was created by English scientist Tim Berners-Lee on August 6, 1991, and it still exists. The page contains information on what the World Wide Web is, how to install a web server, how to download a browser, and other helpful information.

How many people use the Internet?

At the beginning of 2020, more than 4.5 billion people used the Internet. Moreover, the total population of the Earth is estimated at 7.8 billion people, so a significant part of humanity has not yet connected to the Internet.
